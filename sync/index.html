Synchronizacja bazy

Tworzy w Firestore wszystkie kolekcje (strukturę), ustawia Cię administratorem i zapisuje hasło do panelu w osobnej kolekcji.

⚠️ Na czas synchronizacji wklej w Firestore tymczasowe reguły (poniżej), kliknij „Synchronizuj", potem wróć do reguł docelowych.

Zapisze się jako zmienna w Firestore: config/admin → pin. Hasło zmienisz sam, edytując to pole w konsoli Firebase. 

    


    

      1) Wklej TE reguły, zanim klikniesz „Synchronizuj"
      

        
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if request.auth != null;
    }
  }
}

        
        
Firestore → Rules → wklej → Publish. Po synchronizacji wróć do reguł docelowych (bez weryfikacji maila).
